# pms_hotel

ระบบ Hotel PMS (Property Management System) ที่ดีควรมีคุณสมบัติหลักดังนี้:
ด้านการจัดการห้องพัก

ระบบจองห้องและจัดการการเข้า-ออก (Check-in/Check-out) ที่รวดเร็ว
แสดงสถานะห้องแบบ Real-time (ว่าง ไม่ว่าง กำลังทำความสะอาด)
จัดการประเภทห้องและราคาได้ยืดหยุ่น
ระบบ Housekeeping เพื่อติดตามงานแม่บ้าน

ด้านการจองและการขาย

เชื่อมต่อกับ Channel Manager เพื่อจัดการการจองจากหลายช่องทาง (OTA เช่น Booking.com, Agoda)
ระบบ Online Booking Engine สำหรับเว็บไซต์ของโรงแรม
จัดการราคาและโปรโมชั่นตามฤดูกาล

ด้านการเงินและบัญชี

ระบบเก็บเงินและออกใบเสร็จ/ใบกำกับภาษี
บันทึกรายได้รายจ่าย
เชื่อมต่อระบบ POS (ร้านอาหาร, มินิมาร์ท)
รองรับหลายสกุลเงินและหลายช่องทางชำระเงิน

ด้านข้อมูลและรายงาน

ระบบฐานข้อมูลลูกค้า (CRM) เก็บประวัติการเข้าพัก
รายงานสรุปยอดขาย ห้องที่ขายได้ (Occupancy Rate)
Dashboard แสดงข้อมูลสำคัญแบบ Real-time

ด้านเทคโนโลยี

รองรับการใช้งานบน Cloud สามารถเข้าถึงได้ทุกที่
มี Mobile App สำหรับพนักงาน
ระบบรักษาความปลอดภัยข้อมูล (Data Security)
สำรองข้อมูลอัตโนมัติ
